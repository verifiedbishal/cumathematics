1 Problems Involving Sets

Cardinality of a Set

  • What is the total number of element in the word 'M A T H E M A T I C S' ?
  • How many number of elements are there in a set of vowels ?

All these questions refers to the question that explains the number of elements in a given set.

Cardinality of a set is simply the number of element present in any set. For example : The cardinal number of a set of vowels is 5 because there are 5 vowels V = {a,e,i,o,u}

Example 1.1

If n(A) = 10 and n(B) = 20, find the values of n(A ∪ B) and n(A ∩ B) when
a) the sets A and B are disjoint
b) A ⊂ B


Solution

a) When two sets A and B are disjoint, n(A ∩ B) = 0 and
n(A ∪ B) = n(A) + n(B) = 10 + 20 = 30.

b) When A ⊂ B, n(A ∩ B) = n(A) + n(B) - n(A ∪ B) = 10 + 20 - 10 = 20.
